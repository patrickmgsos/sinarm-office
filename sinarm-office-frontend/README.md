# SINARM Office Frontend

Frontend do ERP SINARM Office.

## Direcao Inicial

A interface deve priorizar produtividade operacional, leitura rapida de dados,
seguranca e revisao humana de documentos. O design deve ser sobrio, consistente
e adequado a uso diario por escritorio juridico.

## Stack Prevista

- Bootstrap 5
- HTMX
- Templates server-side quando fizer sentido operacional
- Integracao com APIs do backend quando houver necessidade de desacoplamento

## Principios

- Sem regra de negocio em templates.
- Componentes reutilizaveis.
- Estados de carregamento e erro claros.
- Acessibilidade e responsividade.
- Fluxos otimizados para trabalho repetido.
