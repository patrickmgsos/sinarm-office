# Caso De Uso: Gerar Efetiva Necessidade

## Objetivo

Gerar minuta de efetiva necessidade a partir de modelo versionado e dados do
processo.

## Atores

- Usuario interno autorizado.
- IA assistiva, quando habilitada.

## Fluxo Principal

1. Usuario seleciona processo.
2. Sistema seleciona modelo ativo.
3. Sistema preenche dados estruturados.
4. IA pode auxiliar na minuta.
5. Usuario revisa o conteudo.
6. Sistema gera versao do documento.
7. Sistema registra auditoria.

## Regras

- IA nao toma decisao juridica.
- Revisao humana e obrigatoria.
- Documento deve guardar versao do modelo usado.
