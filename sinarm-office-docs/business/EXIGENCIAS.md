# Exigencias

## Descricao

Exigencia e uma demanda de complementacao, correcao ou esclarecimento emitida
durante a analise de um processo.

## Elementos

- Processo.
- Data de recebimento.
- Descricao.
- Prazo quando houver.
- Documentos exigidos.
- Responsavel.
- Resposta.
- Protocolo de resposta.

## Fluxo De Alto Nivel

1. Registrar exigencia.
2. Definir prazo.
3. Criar tarefas.
4. Ajustar documentos.
5. Revisar resposta.
6. Protocolar resposta.
7. Retornar para analise.

## Regras De Negocio

- Exigencia deve gerar historico.
- Exigencia pendente pode bloquear arquivamento.
- Resposta deve ser auditavel.
- Pode haver multiplas exigencias no mesmo processo.
