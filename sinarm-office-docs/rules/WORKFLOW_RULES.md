# Workflow Rules

## Regras Funcionais

- Workflow nao e status fixo.
- Workflow possui etapas.
- Workflow possui transicoes.
- Processo possui etapa atual dentro de um workflow.
- Mudanca de etapa exige transicao valida.
- Transicao deve gerar historico.
- Transicao pode gerar tarefas e notificacoes.
- Workflow em uso deve ser versionavel.
- Alterar workflow nao deve corromper processos antigos.
- Etapas criticas podem exigir justificativa.
