# Documento Rules

## Regras Funcionais

- Documento nunca e sobrescrito.
- Toda alteracao relevante cria nova versao.
- Documento oficial exige revisao humana.
- Documento gerado deve preservar versao do modelo usado.
- Documento gerado com IA nasce como rascunho.
- IA nao grava documento oficial diretamente.
- Documento assinado nao pode ser editado sem nova versao.
- Documento arquivado permanece consultavel.
- Download de documento sensivel deve ser auditavel.
