# Evento: ProcessoProtocolado

## Quando Ocorre

Quando o protocolo junto a Policia Federal e registrado.

## Dados Minimos

- Processo.
- Numero de protocolo.
- Data de protocolo.
- Usuario responsavel.
- Comprovante quando houver.

## Efeitos Possiveis

- Avancar workflow.
- Registrar auditoria.
- Criar lembrete de acompanhamento.
