# Evento: WorkflowAlterado

## Quando Ocorre

Quando um processo muda de etapa por transicao valida.

## Dados Minimos

- Processo.
- Workflow.
- Etapa anterior.
- Nova etapa.
- Transicao.
- Usuario responsavel.
- Justificativa quando exigida.

## Efeitos Possiveis

- Registrar historico.
- Registrar auditoria.
- Criar tarefas.
- Enviar notificacoes.
