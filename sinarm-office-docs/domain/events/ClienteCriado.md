# Evento: ClienteCriado

## Quando Ocorre

Quando um cliente e cadastrado com sucesso.

## Dados Minimos

- Cliente.
- Empresa.
- Usuario responsavel.
- Data e hora.

## Efeitos Possiveis

- Registrar auditoria.
- Criar pendencias iniciais de documentacao.
- Gerar notificacao interna quando configurado.
