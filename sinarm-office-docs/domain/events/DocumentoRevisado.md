# Evento: DocumentoRevisado

## Quando Ocorre

Quando um usuario humano revisa um documento.

## Dados Minimos

- Documento.
- Versao revisada.
- Usuario revisor.
- Resultado da revisao.
- Data e hora.

## Efeitos Possiveis

- Criar nova versao.
- Aprovar documento.
- Devolver para ajustes.
- Registrar auditoria.
