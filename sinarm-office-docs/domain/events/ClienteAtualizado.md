# Evento: ClienteAtualizado

## Quando Ocorre

Quando dados cadastrais relevantes do cliente sao alterados.

## Dados Minimos

- Cliente.
- Usuario responsavel.
- Campos alterados.
- Valores anteriores quando permitido.
- Valores novos quando permitido.
- Data e hora.

## Efeitos Possiveis

- Registrar auditoria.
- Atualizar pendencias.
- Sinalizar mudancas sensiveis para revisao.
