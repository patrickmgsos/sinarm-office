# Evento: ProcessoCriado

## Quando Ocorre

Quando um processo administrativo e aberto.

## Dados Minimos

- Processo.
- Cliente.
- Tipo do processo.
- Workflow inicial.
- Etapa inicial.
- Usuario responsavel.

## Efeitos Possiveis

- Registrar auditoria.
- Criar tarefas iniciais.
- Criar notificacoes.
