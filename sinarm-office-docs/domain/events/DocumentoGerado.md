# Evento: DocumentoGerado

## Quando Ocorre

Quando um documento e gerado a partir de um modelo versionado.

## Dados Minimos

- Documento.
- Processo.
- Modelo usado.
- Versao do modelo.
- Usuario responsavel.
- Indicacao de uso de IA.

## Efeitos Possiveis

- Registrar auditoria.
- Criar versao inicial.
- Enviar para revisao humana.
