# Evento: ArmaRegistrada

## Quando Ocorre

Quando uma arma e cadastrada e vinculada a um cliente.

## Dados Minimos

- Arma.
- Cliente.
- Usuario responsavel.
- Data e hora.

## Efeitos Possiveis

- Registrar auditoria.
- Criar alertas de validade documental.
- Disponibilizar arma para processos futuros.
