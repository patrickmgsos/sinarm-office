# Compliance

## Proposito

Controlar conformidade operacional, documental, juridica e LGPD.

## Responsabilidades

- Verificar validade de certidoes.
- Detectar documentos vencidos.
- Exigir revisao obrigatoria.
- Apoiar retencao de dados.
- Apoiar politicas de privacidade.
- Sinalizar riscos de conformidade.

## Regras Iniciais

- Documento vencido deve gerar alerta.
- Documento obrigatorio ausente deve gerar pendencia.
- Dados sensiveis devem seguir menor privilegio.
- Retencao deve respeitar politica definida.
- Acoes sensiveis devem ser auditaveis.
