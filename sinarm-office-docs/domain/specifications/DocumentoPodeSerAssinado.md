# Specification: DocumentoPodeSerAssinado

## Pergunta

Documento pode ser assinado?

## Regra Inicial

Documento so pode ser assinado quando estiver revisado e aprovado por usuario
humano autorizado.

## Condicoes Candidatas

- Possui versao atual.
- Nao esta arquivado.
- Revisao humana concluida.
- Modelo usado esta registrado.
