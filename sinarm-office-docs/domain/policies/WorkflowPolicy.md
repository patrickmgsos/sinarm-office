# Policy: WorkflowPolicy

## Objetivo

Controlar quem pode mover processos entre etapas.

## Regras Candidatas

- Usuario deve ter permissao para a transicao.
- Transicoes criticas podem exigir justificativa.
- Transicoes bloqueadas devem ser registradas quando relevante.
- Processo arquivado nao avanca sem reabertura formal.
