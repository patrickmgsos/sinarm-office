# Servico De Dominio: ValidadorSINARM

## Responsabilidade

Centralizar validacoes especificas do dominio SINARM.

## Exemplos

- Validar documentos obrigatorios por tipo de processo.
- Validar necessidade de arma vinculada ao cliente.
- Validar validade de CRAF quando aplicavel.
- Validar pendencias antes de protocolo.

## Regras

- Deve ser configuravel.
- Nao deve depender de telas.
- Deve produzir mensagens auditaveis.
