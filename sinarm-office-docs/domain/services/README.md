# Servicos De Dominio

Servicos de dominio representam operacoes que nao pertencem naturalmente a uma
unica entidade, mas expressam comportamento relevante do negocio.

## Servicos Iniciais

- GeradorDocumentos.
- LegalAutomationEngine.
- WorkflowEngine.
- ValidadorCPF.
- ValidadorSINARM.
- Notificador.
- MotorIA.
