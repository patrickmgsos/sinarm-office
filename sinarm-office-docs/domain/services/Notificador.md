# Servico De Dominio: Notificador

## Responsabilidade

Gerar notificacoes a partir de eventos relevantes.

## Origens

- Workflow.
- Prazos.
- Exigencias.
- Vencimentos.
- Financeiro.

## Regras

- Notificacao critica deve ser rastreavel.
- Notificacao lida nao deve apagar historico.
- Regras de excesso de alertas devem ser configuraveis.
