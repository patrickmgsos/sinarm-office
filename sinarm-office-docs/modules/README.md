# Modulos

Cada modulo deve possuir README proprio, fluxograma, diagrama UML, casos de uso,
regras de negocio, requisitos de seguranca e criterios de aceite.

## Modulos Iniciais

- Accounts
- Clientes
- Armas
- Processos
- Documentos
- Modelos
- IA
- Agenda
- Financeiro
- Relatorios
- Notificacoes
- Auditoria
- Configuracoes

## Template De Documentacao Do Modulo

```text
<modulo>/
  README.md
  use-cases.md
  business-rules.md
  flowchart.md
  uml.md
  security.md
  acceptance-criteria.md
```
