# Documentation Freeze Policy

## Objetivo

Definir como a documentacao sera tratada apos aprovacao da Architecture Review
2.0.

## Regra

Depois da Architecture Review 2.0 aprovada, a documentacao arquitetural sera
considerada congelada para implementacao inicial.

Novas ADRs so deverao ser aceitas quando realmente alterarem decisoes
arquiteturais, regras estruturais ou riscos relevantes.

## Permitido Durante O Freeze

- Correcoes de typo.
- Ajustes de clareza.
- Complementos de implementacao que nao mudam decisao.
- Referencias cruzadas.

## Exige Nova ADR

- Mudanca de chave primaria.
- Mudanca de estrategia de auditoria.
- Mudanca de boundaries de dominio.
- Mudanca de workflow central.
- Mudanca de politica de IA.
- Mudanca de integracao externa relevante.
