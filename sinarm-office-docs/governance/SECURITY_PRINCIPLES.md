# Security Principles

## Principios

- Menor privilegio.
- Segregacao de funcoes.
- Autenticacao forte.
- Autorizacao por perfil e contexto.
- Auditoria de acoes sensiveis.
- Protecao de documentos.
- Protecao contra CSRF, XSS e SQL Injection.
- Backups testados.
- Retencao de dados planejada.
- Conformidade com LGPD.

## Dados Sensiveis

Dados pessoais, documentos juridicos, anexos e historico de processos devem ser
tratados como ativos sensiveis.

## Regra

Toda decisao que exponha, exporte, altere ou exclua dados sensiveis deve ser
rastreavel.
