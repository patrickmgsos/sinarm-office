# Changelog Policy

## Politica

- Toda release deve possuir changelog.
- Mudancas de dominio devem mencionar ADRs ou documentos relacionados.
- Breaking changes devem ser explicitados.
- Correcoes de seguranca devem ser destacadas.

## Categorias

- Added.
- Changed.
- Deprecated.
- Removed.
- Fixed.
- Security.
