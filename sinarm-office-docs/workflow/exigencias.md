# Workflow - Exigencias

## Objetivo

Modelar o subfluxo de exigencias emitidas pela Policia Federal.

## Etapas Candidatas

1. Exigencia recebida.
2. Prazo registrado.
3. Tarefas criadas.
4. Documentos ajustados.
5. Revisao humana.
6. Resposta protocolada.
7. Retorno para analise.

## Regras

- Exigencia deve ter data, descricao e prazo quando houver.
- Pode haver mais de uma exigencia no processo.
- Resposta deve gerar historico e auditoria.
